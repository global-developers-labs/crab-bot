# Crab Bot v2.0 - المحرك الخارق لاختبار الضغط السيبراني

![Crab Bot Logo](assets/crab-bot-logo.png)

## ما هو Crab Bot v2.0؟

**Crab Bot v2.0** هو تطور جذري في أدوات اختبار الضغط (Stress Testing). تم تصميم هذا الإصدار ليكون "خارقاً" بكل ما تحمله الكلمة من معنى، حيث يجمع بين السرعة الهائلة والذكاء في محاكاة السلوك البشري لتجاوز أنظمة الحماية التقليدية واختبار أقصى حدود تحمل الخوادم.

## الميزات الخارقة الجديدة

*   **30,000 بوت ذكي:** نشر جيش من البوتات الافتراضية التي تعمل بالتوازي.
*   **سرعة البرق:** كل 1,000 بوت قادر على إرسال أكثر من 300 طلب في غضون 4 مللي ثانية.
*   **حمولات ذكية (Smart Payloads):** القدرة على إرسال ملفات **JSON** و **XML** عشوائية لمحاكاة طلبات التطبيقات الحقيقية وتجاوز جدران الحماية (WAF).
*   **محاكاة المتصفح:** استخدام رؤوس (Headers) متقدمة لمحاكاة متصفحات Chrome و Safari الحديثة.
*   **تجاوز الكاش:** تقنيات مدمجة لضمان وصول الطلبات إلى الخادم مباشرة دون التأثر بأنظمة الكاش.

## إخلاء مسؤولية (هام جداً)

هذه الأداة قوية جداً وقد تؤدي إلى تعطل الخوادم غير المحمية فوراً. **يُمنع منعاً باتاً** استخدامها ضد أي موقع لا تملك إذناً كتابياً صريحاً لاختباره. الغرض من الأداة هو **التعليم والبحث الأمني فقط**.

## التثبيت السريع

```bash
# تأكد من وجود Node.js مثبت على جهازك
npm install -g crab-bot
```

## كيفية الاستخدام

ببساطة اكتب الأمر التالي في الترمينال واتبع التعليمات التفاعلية:

```bash
crab-bot
```

ستطلب منك الأداة:
1.  رابط الموقع المستهدف.
2.  نوع الحمولة (JSON, XML, أو عشوائي).
3.  تأكيد البدء.

## مثال على الأداء

| الميزة | القدرة |
| :--- | :--- |
| عدد البوتات | 30,000 |
| الطلبات لكل بوت | 300+ |
| الفاصل الزمني | 4ms |
| أنواع البيانات | JSON / XML / HTTP POST |

## المساهمة

المشروع مفتوح المصدر بالكامل. نرحب بالمطورين والباحثين الأمنيين للمساهمة في تحسين المحرك وإضافة تقنيات تجاوز جديدة.
