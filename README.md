# Crab Bot - أداة اختبار الضغط السيبراني

![Crab Bot Logo](https://raw.githubusercontent.com/CrabBot-Dev/CrabBot/main/assets/crab-bot-logo.png) <!-- Placeholder for a future logo -->

## مقدمة

**Crab Bot** هي أداة قوية ومفتوحة المصدر مصممة لاختبار مرونة المواقع والتطبيقات ضد هجمات حجب الخدمة الموزعة (DDoS). تم تطوير هذه الأداة خصيصًا لأغراض **تعليمية وبحثية**، ولتمكين مطوري الويب وخبراء الأمن السيبراني من تقييم قوة وصلابة أنظمتهم الدفاعية في بيئة خاضعة للرقابة. تحاكي الأداة هجومًا حقيقيًا باستخدام ما يصل إلى 30,000 "بوت" افتراضي لإرسال طلبات متزامنة إلى الهدف المحدد.

## إخلاء مسؤولية وتحذير هام

**Crab Bot مخصصة للاستخدام القانوني والأخلاقي فقط.** يجب استخدام هذه الأداة **فقط** على الأنظمة التي تملك صلاحية صريحة وكاملة لاختبارها. أي استخدام غير مصرح به لهذه الأداة ضد أنظمة لا تملك الحق في اختبارها يعتبر **غير قانوني** وقد يؤدي إلى عواقب وخيمة، بما في ذلك الملاحقة القانونية. المطورون لا يتحملون أي مسؤولية عن أي استخدام غير قانوني أو غير أخلاقي لهذه الأداة.

**استخدم هذه الأداة بمسؤولية وعلى مسؤوليتك الخاصة.**

## الميزات

*   **محاكاة DDoS واقعية:** توليد ما يصل إلى 30,000 طلب HTTP متزامن لمحاكاة هجوم DDoS حقيقي.
*   **واجهة سطر الأوامر (CLI):** سهولة الاستخدام عبر واجهة تفاعلية في الطرفية.
*   **أداة تعليمية:** مثالية لفهم آليات هجمات DDoS وكيفية الدفاع ضدها.
*   **مفتوحة المصدر:** يمكن للمجتمع مراجعة الكود والمساهمة في تطويره.

## التثبيت

تتطلب Crab Bot تثبيت [Node.js](https://nodejs.org/) (الإصدار 14 أو أحدث) على نظامك.

1.  **تثبيت Node.js (إذا لم يكن مثبتًا):**
    قم بزيارة [الموقع الرسمي لـ Node.js](https://nodejs.org/) وقم بتنزيل وتثبيت الإصدار الموصى به لنظام التشغيل الخاص بك.

2.  **تثبيت Crab Bot عالميًا:**
    افتح الطرفية (Terminal/Command Prompt) وقم بتشغيل الأمر التالي:

    ```bash
    npm install -g crab-bot
    ```

    قد تحتاج إلى استخدام `sudo` قبل الأمر في أنظمة Linux/macOS:

    ```bash
    sudo npm install -g crab-bot
    ```

## الاستخدام

بعد التثبيت، يمكنك تشغيل Crab Bot ببساطة عن طريق كتابة `crab-bot` في الطرفية:

```bash

```

ستطالبك الأداة بإدخال عنوان URL للهدف الذي ترغب في اختباره، ثم ستطلب منك تأكيد بدء اختبار الضغط.
